### Kurulum Adımları

1. Projeyi bilgisayarınıza indirin:

```bash

git  clone [https://github.com/AlperenlCan/AutoCare.git]

```

  

2. Proje klasörüne gidin:

```bash

cd [AutoCare]

```

  

3. Gerekli paketleri yükleyin:

```bash

flutter  pub  get

```

  

4. Uygulamayı başlatın:

```bash

flutter  run

```
